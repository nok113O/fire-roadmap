# FIREロードマップ

FIRE(経済的自立と早期リタイア)達成までの資産推移をシミュレーションし、毎月の実績を記録・振り返るための Web アプリです。React + TypeScript + Vite で構築されています。

## できること

- 現在の年齢・資産額(日本円/人民元の内訳+為替レート)・毎月の貯蓄額・想定利回り・FIRE後の年間支出・安全引出率(SWR)から、FIRE達成に必要な資産額と達成予定時期を計算
- 資産推移の計画ラインをグラフで表示
- 毎月の実際の資産額(日本円/人民元の内訳+その月の為替レート)を記録し、計画との差分・進捗率を確認(グラフにも実績を重ねて表示)
- 過去分の実績をまとめてテキスト貼り付けで一括登録可能(「年月,日本資産(万円),中国資産(元),為替レート」形式)
- 入力値・記録内容はブラウザの localStorage にのみ保存され、リポジトリには含まれません(次回アクセス時も同じブラウザなら継続して利用可能)

