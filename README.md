# FIREロードマップ

FIRE(経済的自立と早期リタイア)達成までの資産推移をシミュレーションし、毎月の実績を記録・振り返るための Web アプリです。React + TypeScript + Vite で構築されています。

## できること

- 現在の年齢・資産額・毎月の貯蓄額・想定利回り・FIRE後の年間支出・安全引出率(SWR)から、FIRE達成に必要な資産額と達成予定時期を計算
- 資産推移の計画ラインをグラフで表示
- 毎月の実際の資産額を記録し、計画との差分・進捗率を確認(グラフにも実績を重ねて表示)
- 入力値・記録内容はブラウザの localStorage に保存されるため、次回アクセス時も継続して利用可能

