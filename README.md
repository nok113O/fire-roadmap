# FIREロードマップ

FIRE(経済的自立と早期リタイア)達成までの資産推移をシミュレーションし、毎月の実績を記録・振り返るための Web アプリです。React + TypeScript + Vite で構築されています。

## できること

- 現在の年齢・資産額(日本円/人民元の内訳+為替レート)・毎月の貯蓄額・想定利回りから、セミFIRE(年間支出−就労収入をSWRで割った必要資産額)と完全FIRE(年間支出をSWRで割った必要資産額)それぞれの必要資産額・達成予定時期を計算。SWRは目標ごとに個別設定可能
- 資産推移の計画ラインと、セミFIRE/完全FIREそれぞれの必要資産額のラインをグラフで表示
- 毎月の実際の資産額を、日本側は自由に追加・編集できる口座別残高(万円)、中国側は資産額(元)として記録し、その月の為替レートで円換算した合計を計画と比較(グラフにも実績を重ねて表示)
- 日本・中国それぞれの毎月の収入・支出も記録し、貯蓄率を自動計算
- 過去分の実績をまとめてテキスト貼り付けで一括登録可能(「年月,日本資産合計(万円),中国資産(元),為替レート」形式。口座別の内訳は付かず合計のみ)
- 家族メンバー(生年月)を登録し、教育費(小学校〜大学)などのライフイベントの開始・終了時期を自動計算。住宅購入や車の買い替えなどの一時的な支出・継続的な支出も登録でき、資産推移の計画に反映
- 入力値・記録内容はブラウザの localStorage にのみ保存され、リポジトリには含まれません(次回アクセス時も同じブラウザなら継続して利用可能)

